Used or adapted concepts:
- v2 light theme token structure from packages/ui/src/v2/styles/colors.css
  and packages/ui/src/v2/styles/theme.css
- Sidebar rail plus expandable session panel from packages/app/src/pages/layout
- Session timeline row model from packages/app/src/pages/session
- Prompt dock layout and reserved attachment/model controls from
  packages/app/src/components/prompt-input.tsx
- Auto-follow scroll behavior based on packages/ui/src/hooks/create-auto-scroll.tsx

Pando-specific boundaries:
- Backend API remains Pando's local /api routes.
- No OpenCode SDK, server-sync, provider auth, desktop shell, or workspace
  backend code is copied into Pando in this step.

# Agent Instructions

1. Do not create unnecessary entities.
2. Be factual. If something is unknown or unsafe, say so and stop instead of guessing.
3. Use English and ASCII names for files, directories, program identifiers, config keys, and script paths by default.
4. Use Chinese only for semantic content that needs Chinese, such as prompts, recognition text, display copy, or explicit user-requested content.
5. Keep architecture close to the local Claude Code source study material, but do not copy proprietary or decompiled implementation details.
6. GUI automation is a first-class execution path, not the only foundation. Prefer deterministic tools first, then GUI when the real application surface is required.
